Was ist WebTigerPython?

WebTigerPython ist eine webbasierte Entwicklungsumgebung für die Programmiersprache Python. Sie wurde speziell für den Einsatz im Schulunterricht entwickelt und ermöglicht es, Python-Code direkt im Browser zu schreiben und auszuführen – ohne Installation. Dies macht es besonders einfach, erste Schritte im Programmieren zu unternehmen.

import math
from gpanel import *

def step():
    global t
    x = r * math.cos(t)
    y = r * math.sin(t)
    t = t + 0.2
    if t > 0:
        t = 0
        setColor(getRandomX11Color())
    move(x, y)
    fillCircle(-0.1)

makeGPanel(-300, 1000, 200, -300)
bgColor("green")
t = 0
r = 200
while True:
    step()
